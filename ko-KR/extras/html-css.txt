### HTML이란?

HyperText Markup Language (HTML) 은 웹페이지를 설명하고 짓는 언어입니다. HTML 문서는 텍스트, 이미지, 소리와 비디오를 `tags`를 통해 정리하여 넣을 수 있습니다. 다음은 헤딩 (`<h1>`), a paragraph of text (`<p>`) 와 이미지(`<img>`)를 포함한 간단한 HTML 의 예입니다: 

![screenshot](01/html-example.png)

### CSS란?

Cascading Style Sheets (CSS) 은 HTML 문서를 구성하고 스타일 하는데 사용하는 또 다른 언어입니다. 예를 들자면 위의 paragraph of text 를 이 CSS 코드로 파란색으로 만들 수 있습니다:

![screenshot](01/css-example.png)

### 트링켓 사용하기

저희들의 HTML & CSS 프로젝트를 작업할 땐 [트링켓](https://trinket.io)을 사용하여 웹페이지를 온라인으로 만드는 걸 추천 드립니다. 트링켓은 오프라인으로 텍스트 에디터를 사용하는 것에 비해 다음과 같은 장점이 있습니다:

+ 아이들이 실시간으로 작업중인 HTML & CSS 코드의 미리보기를 볼 수 있습니다;
+ 아이들이 온라인으로 코드를 저장하여 웹페이지를 클럽에서도 집에서도 볼 수 있습니다;
+ 아이들이 다른 이들과 웹페이지를 공유하고 같이 재구성할 수 있습니다.

첫번째 HTML & CSS 프로젝트 ([생일 축하 합니다](http://jumpto.cc/birthday)) 에서 간단한 웹페이지를 만들고 저장하고 보는 방법을 좀 더 자세히 알려줍니다.

### 오프라인으로 작업하기

클럽의 아이들이 오프라인으로 HTML & CSS 코드를 작성해야 할 때가 있을지도 모릅니다. 다음은 HTML & CSS를 오프라인으로 쓰고, 수정하고, 보는 방법에 대한 짧은 가이드 입니다.

###### 쓰기

각 프로젝트의 다운로드 받을 수 있는 소재들은 다음 파일들을 포함하고 있습니다:

+ Example.html
+ style.css
 
이 파일들(과 포함된 이미지들)이 같은 폴더에 있는지 확인하세요. (직접 HTML을 쓰고 싶다면 각 HTML 프로젝트의 'Project Resources'에서 템플릿 웹페이지 문서를 다운로드 받을 수 있습니다)

파일을 두개 다 텍스트 에디터에서 여세요. 저희들은 [Sublime Text 3](http://www.sublimetext.com/3)를 추천하지만 다른 좋은 에디터들도 매우 많습니다. `.html`과 `.css`파일을 동시에 여는 가장 쉬운 방법은 양쪽 파일 둘다 하이라이트 시켜서 오른쪽 클릭 하여 텍스트 에디터로 여는 것입니다.

![screenshot](01/offline-open.png)

###### 저장하기

HTML & CSS를 쓴 후 파일을 저장하세요- 파일들이 `.html` 과 `.css` 파일 확장자인걸 확인하세요.

![screenshot](01/offline-save.png)

###### 보기

웹페이지를 보려면  `Example.html`파일을 더블 클릭하여 브라우저에서 여세요.

![screenshot](01/offline-view.png)

만약 HTML 파일이 텍스트 에디터에서 열린다면 `.html` 확장자 인지 확인하세요.

차이점을 보는 가장 쉬운 방법은 파일들을 브라우저 _와_ 텍스트 에디터로 동시에 여는 것입니다. 텍스트 에디터에 변화를 준 후 저장하고 브라우저의 '새로 고침' 버튼을 눌러서 바뀐 점을 볼 수 있습니다.

![screenshot](01/offline-edit.png)
